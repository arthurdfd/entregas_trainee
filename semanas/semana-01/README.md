# Semana 1: Fundamentos de Python & Git

Nesta primeira semana do programa Trainee, a trilha de aprendizado foi focada em consolidar as bases da linguagem Python e estabelecer boas práticas de controle de versão utilizando Git e GitHub.

---

### 1. Introdução ao Ambiente & Git
- Configuração do ambiente local de desenvolvimento no VS Code.
- Conceitos essenciais de versionamento: fluxo de trabalho profissional com comandos `git add`, `git commit` estruturados e `git push`.

### 2. Sintaxe e Estruturas de Controle (Python)
- Declaração de variáveis, tipagem dinâmica e operações aritméticas/lógicas.
- Condicionais estruturadas (`if`, `elif`, `else`) para controle de fluxo.
- Laços de repetição (`for` e `while`) para iterações eficientes.

### 3. Estruturas de Dados Avançadas (Nível 3 - Data Analytics)
- **Listas e Tuplas:** Manipulação, fatiamento (*slicing*) e métodos nativos de ordenação e inserção.
- **Dicionários (Dicts):** Estruturas de chave-valor para mapeamento de dados complexos.
- **Conjuntos (Sets):** Otimização de buscas e eliminação de duplicatas operando em tempo constante $O(1)$.
